videojuego

La etiqueta video: para especificar la ubicación del archivo que se reproducirá, puede usar el elemento <source> dentro del elemento <video>. (Esto debe reemplazar el atributo src en la etiqueta <video> de apertura.) También puede utilizar varios elementos <source> para especificar que el video está disponible en diferentes formatos. (Debido a un error en el ipad, debe proporcionar el video MP4 como el primer formato. De lo contrario, puede que no inicie.)

src: este atributo especifica la ruta de acceso al vídeo.

type: se debe utilizar este atributo para decirle al navegador qué formato es el vídeo. De lo contrario, descargará parte del video para ver si puede reproducir el archivo (que tomará tiempo y ancho de banda).

codecs: el códec que se usó para codificar el video se suministra dentro del atributo type. Tenga en cuenta el uso de las comillas simples, así como comillas dobles en el atributo de tipo, cuando se suministra
